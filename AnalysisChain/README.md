# MatlabProcFM

## Usage Instructions

The FM passive radar processing chain accepts inputs in the form of RCF or HDF5 files. This has been tested using HDF5 files generated by the PXGFToMatlab application.

### RCF Files

* Set `hdf5_input` to `0`
* Provide the RCF filename in `InputRCFFilename`
* Specify the desired cancellation and ARD processing parameters
* Run the script
* The ARD files can be viewed in ARDView (tested with V0.2.6)

### HDF5 Files

* Generate a HDF5 file from a PXGF using the PXGFToMatlab application
* Set `hdf5_input` to `1`
* Provide the HDF5 filename in `InputHDF5Filename`
* Specify the desired cancellation and ARD processing parameters
* Run `whos('-file', InputHDF5Filename)` to see which GSNC chunks are stored in the file and select one by replacing `X` with the chunk number in the following lines
  * `load(InputHDF5Filename, 'GSNC_dataX', '-mat');`
  * `RefData = double(GSNC_dataX(:,RefChannelIndex));`
  * `SurvData = double(GSNC_dataX(:,SurvChannelIndex));`
  * `clear GSNC_dataX;`
* Run the script
* The ARD files can be viewed in ARDView (tested with V0.2.6)
