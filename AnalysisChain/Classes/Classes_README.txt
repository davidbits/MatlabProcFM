This folder contains matlab class files that define objects of common data types

RCF - Raw Capture Format (a block of raw data).
ARD - Amplitude/Range/Doppler map.
ARDCell - A single cell from a ARD map.
ARDCellSelection - A collection of ARDCells from a ARD map.

DataReader.class and .java is java code for reading data from sockets. It is called inline in the Matlab classes.
