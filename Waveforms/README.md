[jammerWaveFormNormalised.h5](jammerWaveFormNormalised.h5) uses the [Malmesbury_1.rcf](Malmesbury_1.rcf) file to generate a normalised jammer waveform.
The waveform is a normalised snippet from 540s to 720s of the original waveform.

[txWaveFormNormalised.h5](txWaveFormNormalised.h5) uses the [Malmesbury_1.rcf](Malmesbury_1.rcf) file to generate a normalised transmitter waveform.
The waveform is a normalised snippet from 360s to 540s of the original waveform
